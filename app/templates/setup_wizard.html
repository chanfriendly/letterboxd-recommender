Connect to the movie database

Recommendations are powered by TMDB — a free, community-built movie database. Getting an API key takes about 2 minutes and costs nothing.

How to get your free API key

  1. 1. Go to themoviedb.org/signup and create a free account (or log in if you have one)
  2. 2. Visit themoviedb.org/settings/api
  3. 3. Click Create under "API Key (v3 auth)" — choose Personal use when asked
  4. 4. Copy the long key that appears and paste it below

Import your Letterboxd history

We need your watch history to generate personalised recommendations. This is a one-time import — new watches sync automatically every 6 hours after that.

How to export your Letterboxd data

  1. 1. Log in to Letterboxd and go to letterboxd.com/settings/data/
  2. 2. Click Export Your Data — Letterboxd will email you a download link within a few minutes
  3. 3. Download the .zip file from the email — don't unzip it
  4. 4. Fill in the form below and upload the .zip
