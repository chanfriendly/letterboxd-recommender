Setup

Add each person's Letterboxd profile. Import their full watch history by uploading a Letterboxd data export — ratings and watched films are picked up automatically. New entries sync via RSS every 6 hours after that.

How to get your Letterboxd data export

  1. Go to letterboxd.com/settings/data/ (must be logged in)
  2. Click Export Your Data and wait for the email
  3. Download the .zip file from the email link
  4. Upload it below — no need to unzip

Only the username is stored after import. The ZIP file is deleted immediately after processing.

Deep Semantic Matching

Uses a local AI model (all-MiniLM-L6-v2) to read the plot description of every film in your library and understand thematic meaning — not just genre labels. Once enabled, the app can follow taste throughlines across genres: if you love films about moral ambiguity under pressure, it will find others with that same undercurrent even if they're tagged differently. Runs entirely on your server; no data leaves your machine.

Requires sentence-transformers to be installed in the container (pip install sentence-transformers). The first run embeds all films in your library — a few minutes depending on library size. New films are embedded automatically after that.
